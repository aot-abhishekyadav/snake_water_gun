# snake_water_gun
# Snake Water Gun Game 🐍💧🔫

A simple and fun command-line based Snake Water Gun game built using Python.  
This project uses Python fundamentals like conditional statements, dictionaries, user input handling, and random number generation to simulate gameplay between the user and the computer.

---

## 📌 About the Project

Snake Water Gun is similar to the classic Rock Paper Scissors game.

In this game:
- Snake drinks Water → Snake Wins
- Water drowns Gun → Water Wins
- Gun kills Snake → Gun Wins

The computer randomly selects a choice, and the user enters their move through the terminal.

---

## 🚀 Features

- Interactive terminal gameplay
- Random computer choice generation
- Win / Lose / Draw result system
- Beginner-friendly Python project
- Clean and simple game logic

---

## 🛠️ Technologies Used

- Python 3
- Random Module
